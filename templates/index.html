Health Tips API Documentation

Github

This API provides health tips based on different parameters like mood, temperature change, working out time, physical activity change, heart rate change, and sleep change.

Endpoints

  • /mood
    • GET: /mood/stormy
    • GET: /mood/anxiety
  • /temperature
    • GET: /temperature/increase
    • GET: /temperature/decrease
  • /working-out-time
    • GET: /working-out-time/increase
  • /physical-activity
    • GET: /physical-activity/decrease
  • /heart-rate
    • GET: /heart-rate/improve
    • GET: /heart-rate/decrease
  • /sleep-time
    • GET: /sleep-time/increase
    • GET: /sleep-time/reduce

    Response Format

    {"health_tip": string}

    Error Response for invalid endpoint (Status: 404)

    {"error": string}
